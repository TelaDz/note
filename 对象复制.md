# 对象复制

## for in

```js
fn1();
function fn1(){
let obj={a:1,b:2,c:[2,3,4],d:{a:1,b:2,c:{a:1,b:2,d:new Date()}}};
let obj1={};
for(var prop in obj)obj1[prop]=obj[prop];
console.log(obj1);
}
```

## JSON

```js
fn2();
function fn2(){
```
