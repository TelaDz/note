# H5

## 简化的文档类型和字符集

### 文档类型

<!DOCTYPE HTML>

之所以如此简单，是因为 HTML5 不再是 SGML（ Standard Generalized Markup language，标准通用标记语言）的一部分，而是独立的标记语言，不需要再考虑文档类型

### 字符集

`<meat charset="UTF-8">`只需要 utf-8 即可
